#  Premium Sports Center PWA

![Titan Sports](https://images.unsplash.com/photo-1534438327276-14e5300c3a48?q=80&w=1200&auto=format&fit=crop)

Una solución integral y de alta gama para la gestión de centros deportivos (Gimnasio, Alberca y Gimnasia Olímpica). Diseñada con una estética moderna **"Cyber-Luxury"** y funcionalidades de vanguardia.

##  Características Principales

-  **PWA (Progressive Web App):** Instalable en dispositivos iOS y Android.
-  **Modo Offline:** Acceso a la información esencial incluso sin conexión a internet.
-  **Diseño Premium:** Interfaz basada en Glassmorphism con acentos neón y animaciones fluidas.
-  **Simulación Biométrica:** Sistema de acceso simulado mediante FaceID/QR para una experiencia de usuario moderna.
- 👤 **Portal del Cliente:**
  - Consulta de horarios de clases.
  - Gestión de notificaciones y avisos.
  - Carga de comprobantes de pago.
  - Reporte de ausencias automático.
-  **Panel de Administración:**
  - Estadísticas en tiempo real de usuarios y pagos.
  - Sistema de aprobación/rechazo de pagos.
  - Publicación de avisos globales.

##  Tecnologías Utilizadas

- **Frontend:** HTML5, Vanilla CSS3 (Custom Properties), JavaScript ES6.
- **Persistencia:** LocalStorage API para almacenamiento local de datos.
- **PWA:** Service Workers y Web App Manifest.
- **Diseño:** Google Fonts (Inter) y FontAwesome.

##  Instalación y Uso Local

1. Clona este repositorio o descarga los archivos.
2. Abre la carpeta en tu editor de código.
3. Para Windows, puedes ejecutar `start_server.bat` para iniciar el servidor local.
4. Accede a `http://localhost:8000`.
